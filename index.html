How to use this tool?

    EPIC
  • Open "Physiologic Monitoring Data from Anesthesia Record Report (from chart review
  • Select all (Ctrl + A)
  • Copy (Ctrl + C)
  • Open a blank/new text document in notepad
  • paste (ctrl+v) copied data into the text file
  • Save file as whatever-filename.txt
    Web Portion
  • Choose whatever-filename.txt
  • Check the boxes of the vital signs you need extracted
  • Upload and wait
  • Depending on the "size" of the request- wait time may vary
  • You will either be prompted to "save as" or if using chrome a file will just get downloaded
  • File name will whatever-filename.txt.csv
    Excel or anyother spreadsheet
  • Open whatever-filename.txt.csv directly into Excel
  • Or if having encoding issues - Open Excel and go to Import data from Text and then import as Unicode UTF-8
  • Sort or Filter your extracted vital sign values, units and timestamps and use it in your analysis
